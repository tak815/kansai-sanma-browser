# v0.3.14-1 リプレイ基盤

## 確認手順
1. 通常どおり数巡進めます。
2. 「リプレイ」を押します。
3. 「前へ」「次へ」でツモ・打牌単位に表示が変わることを確認します。
4. 「再生」でイベントが自動的に進むことを確認します。
5. ページを再読み込みしても、最後に保存された牌譜を開けることを確認します。

## 保存内容
局開始、ツモ、打牌、リーチ打牌、ポン・カン・ロン・ツモ等の成立表示、流局、対局終了をイベント列として保存します。各イベントには手牌、河、副露、抜き花、ドラ表示、点棒、供託、山残りのスナップショットが含まれます。

## 将来拡張
オンライン観戦ではこのイベント形式をサーバーに保存し、高レート対局を一定時間遅延させて配信します。対局中の観戦者に未公開情報が即時送信されない構造を前提とします。
