Project KSB v0.3.12-2 自動手番進行

実装内容
- 局開始時、親のツモを自動実行
- 打牌後、次家のツモを自動実行
- 南家・西家はツモ後に既存CPU思考で自動打牌
- 自家へ戻ったときも自動ツモし、打牌選択待ちで停止
- 「ツモ / 進行」は通常不要な予備ボタンとして残す
- ロン・ツモ和了・ポン・カンなどの割り込み中は自動ツモしない
- AI停止中はCPU手番だけ保留し、自家手番は通常どおり進行

確認方法
1. 「現在局を作り直す」を押す
2. 親のツモが自動で行われることを確認
3. 自家の打牌後、南家・西家が自動でツモ打牌することを確認
4. 自家へ戻った時に自動で14枚になり、打牌待ちになることを確認
5. AI停止中はCPU手番で止まり、AI再開後に進むことを確認
