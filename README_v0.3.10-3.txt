Project KSB v0.3.10-3
Sprint 3-10-3: 本場・連荘・流局基盤

追加内容:
- 流局時のテンパイ/ノーテン判定マップを追加
- 親テンパイなら連荘、本場+1
- 親ノーテンなら親流れ、本場+1
- 流局時の供託は次局へ持越し
- ノーテン罰符の暫定点棒移動を追加
  - 1人聴牌: 聴牌者+3000、ノーテン2人-1500ずつ
  - 2人聴牌: 聴牌2人+1500ずつ、ノーテン1人-3000
  - 全員聴牌/全員ノーテン: 点棒移動なし
- デバッグボタン追加: 親テンパイ流局 / 親ノーテン流局

確認ポイント:
1. 親テンパイ流局 → 同じ局のまま本場+1、親継続
2. 親ノーテン流局 → 次局へ進み、親が次家へ移動、本場+1
3. リーチ供託がある場合、流局後も次局へ持ち越される
4. ノーテン罰符で点棒が変化する
