Project KSB v0.3.15-2 通常対局操作UI仕上げ

追加・変更
- 通常表示では、その時点で押せる主要操作ボタンのみ表示
- 「捨てる牌を選んでください」「自分の手番までお待ちください」などの案内を追加
- ロン・ツモ・リーチ・カンなど、成立可能な操作を見つけやすく調整
- スマホ横画面で操作ボタンの高さと押しやすさを改善
- 開発表示では従来どおり全ボタンを確認可能

確認
1. 通常表示で無効な操作ボタンが消える
2. 自分の打牌待ちでは「捨てる牌を選んでください」と表示される
3. リーチ・ロン・ツモなどが可能なときだけ該当ボタンが表示される
4. 開発表示へ戻すと確認用ボタンを含む従来表示になる
