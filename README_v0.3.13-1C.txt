Project KSB v0.3.13-1C

演出の表示タイミング修正です。

確認手順
1. リーチ可能な状態で「リーチ」を押す。
   - この時点では供託・点棒・文字演出は変化しません。
   - ボタンは「リーチ取消」になります。
2. 「リーチ取消」を押す。
   - リーチ選択だけが解除されます。
3. 再度リーチを選び、宣言牌を打つ。
   - この瞬間に「リーチ」が表示されます。
   - 1000点減少、供託+1、宣言牌横向きが同時に確定します。
4. CPUポン確認で發を捨てる。
   - ポンが実際に成立した瞬間だけ「ポン」が表示されます。
5. カン不可状態では「カン」が表示されないことを確認します。
