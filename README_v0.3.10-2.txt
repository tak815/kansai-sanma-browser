Project KSB v0.3.10-2
Sprint 3-10-2: Next hand foundation

変更内容:
- 局終了後に「次局へ」ボタンを追加
- 和了後の親継続/親流れの基盤を追加
- 流局時は暫定で親継続+本場+1として処理
- 点棒を次局へ引き継ぐ
- 和了時は供託を和了者へ加算後、次局供託0
- 流局時は供託を次局へ持ち越し
- 局表示を現在の roundWind / handNumber から更新

確認項目:
1. リーチ確認→リーチ→ロン/ツモ確認などで局終了後、「次局へ」が押せる。
2. 親（自家）が和了した場合、東1局のまま1本場になる。
3. 子和了のデバッグ確認では親が次家へ移る。
4. 点棒が35000に戻らず次局へ引き継がれる。
5. 供託は和了者へ加算され、次局では0になる。
