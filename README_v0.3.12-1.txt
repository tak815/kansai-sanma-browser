Project KSB v0.3.12-1 対局進行修正

実装内容
- 東場は東1局→東2局→東3局まで進行
- 東3局終了後は南1局へ移行
- 南場は南1局→南2局→南3局まで進行
- 南3局終了時、誰も40000点を超えていなければ返り東1局へ移行
- 返り東では、局終了時に誰かが40000点を超えた時点で対局終了
- 返り東で条件未達の場合は東1〜3局を循環
- 画面とログでは「返り東 東1局」のように通常東場と区別

今回未実装
- オーラス親の上がりやめ選択UI
- 箱下・飛ばし賞の完全精算
- ダブロン精算

確認方法
1. デバッグの南3局移動を利用する
2. 全員40000点以下で南3局の親を流す
3. 次局表示が「返り東 東1局」になることを確認
4. 返り東で誰かを40001点以上にして局を終了
5. 対局終了になることを確認
