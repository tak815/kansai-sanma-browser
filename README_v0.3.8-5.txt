Project KSB v0.3.8-5 / Sprint 3-8-5
カン基盤

変更内容:
- 暗槓の実行基盤を追加
- 加槓の実行基盤を追加
- カン成立時にカンドラ表示牌を1枚追加
- カン成立時にカン裏ドラ候補を1枚確保
- カン成立時、表示上の山残りを2枚減少
- 嶺上牌から補充ツモ
- カンドラを新たにめくれない場合はカン不可
- 花牌は空気牌のためカン対象外
- カン不可確認用デバッグを追加

確認方法:
1. デバッグ暗槓を押す
2. カンボタンが出ることを確認
3. カンを押す
4. 5p暗槓が副露/カン表示へ移動
5. ドラ表示牌が1枚増える
6. 山残りが2枚減る
7. 嶺上牌が1枚減る
8. 嶺上ツモ後に打牌できる

加槓確認:
1. デバッグ加槓を押す
2. カンボタンを押す
3. ポン表示が加槓表示になる
4. ドラ・山・嶺上の減少が暗槓と同じになる

カン不可確認:
1. カン不可確認を押す
2. 5pが4枚あってもカンボタンが出ないことを確認
