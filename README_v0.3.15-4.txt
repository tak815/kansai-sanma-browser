Project KSB v0.3.15-4

変更内容
- 通常画面の南家・西家の河を、各家から見て正面になる向きへ回転
- 河は7枚×3段のグリッドで表示
- リーチ宣言牌だけ河の並びを崩さず90度回転
- 手牌に花牌があり抜き花可能なとき、手牌直下に「抜き花」ボタンを常時表示
- 抜き花可能な花牌を淡く強調表示
- 花牌は捨て牌として選べない既存仕様を維持

確認
1. 通常表示で南家の河が右席向き、西家の河が左席向きになる
2. 河が7枚ごとに折り返す
3. リーチ牌だけ横向きになる
4. 自家手牌に花牌があると「抜き花」が手牌直下に表示される
5. 抜き花後に花牌が抜き花欄へ移動し、嶺上補充される
