Project KSB v0.3.13-2 効果音基盤

追加内容
- Web Audio APIによる軽量な効果音（外部音声ファイル不要）
- ツモ、打牌、リーチ、ポン、カン、ロン、ツモ和了、抜き花、カンドラ追加
- 効果音ON/OFFボタン
- リーチやポン等は、選択時ではなく実際に成立した時だけ再生

確認
1. 効果音 ON の状態で通常ツモ・打牌を行う
2. リーチ確認で宣言牌を実際に打った時だけ音が鳴る
3. ロン、ツモ、カン、抜き花の成立時にそれぞれ音が鳴る
4. 効果音 OFF で音が停止する

ブラウザ仕様により、最初の音は画面上のボタンや牌を一度操作した後に有効になります。
