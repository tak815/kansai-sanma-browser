Project KSB v0.3.13-6 統計イベント・役別分析基盤

追加内容
- 統計保存形式を schemaVersion 2 へ更新
- 旧バージョンの保存データを自動移行
- 和了履歴を最大500件の個別イベントとして保存
- 役ごとのロン回数・ツモ回数を保存
- 役ごとの平均翻・平均打点を表示
- winHistory と extensionData により将来のプレミアム分析項目を追加可能

確認
1. 旧統計データが残っていても統計画面が開く
2. 和了後に対局を終了すると役別統計が更新される
3. 役別表に和了回数、ロン、ツモ、平均翻、平均打点が表示される
4. JSON書き出しで schemaVersion が 2、winHistory が存在する
