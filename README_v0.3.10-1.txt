Project KSB v0.3.10-1

点棒支払い計算の基盤を追加しました。

追加内容:
- 点棒表示（自家 / 南家 / 西家）
- リーチ宣言時に自家点棒から1000点を供託へ移動
- ロン和了時の支払い反映
- ツモ和了時の親/子支払い反映
- 供託の和了者加算
- 本場加算の基盤（ロン+300、ツモ各+100）

確認ポイント:
- リーチ宣言で自家点棒が1000減り、供託が1になる
- ロン和了で放銃者から和了者へ点棒移動する
- ツモ和了で他2家から支払いが発生する
- 供託が和了者へ加算され、和了後に0へ戻る

注意:
- 満貫未満の符計算は後続Sprintで実装予定です。
- 連荘、局送り、終局処理は次以降で実装予定です。
