Project KSB v0.3.6A
HTML + CSS + JavaScript 版

今回の実装:
- React / TypeScript / Vite は使用していません。
- index.html 一体型の試作構成です。
- 暗槓の判定と実行を追加。
- 加槓の判定と実行を追加。
- 嶺上ツモの土台処理を追加。
- カン可能時だけ「カン」ボタンを表示。
- 副露 / カン表示エリアを追加。
- デバッグ暗槓、デバッグ加槓ボタンを追加。

確認方法:
1. ZIPを解凍します。
2. 既存プロジェクトの index.html をバックアップします。
3. この index.html を上書きします。
4. ブラウザで index.html を開きます。
5. 「デバッグ暗槓」→「カン」で暗槓と嶺上ツモを確認します。
6. 「デバッグ加槓」→「カン」で加槓と嶺上ツモを確認します。

注意:
- カンドラ、裏ドラ、槍槓、嶺上開花の役判定、AI判断は未実装です。
- 今回はカン機能の土台確認用です。
- 画像牌は既存の images フォルダをそのまま使います。画像がない場合は文字牌で表示されます。
