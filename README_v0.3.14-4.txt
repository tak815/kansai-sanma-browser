Project KSB v0.3.14-4 観戦プライバシー・ディレイ基盤

追加内容
- リプレイ画面に観戦表示モードを追加
  - 全手牌
  - 自家のみ
  - 全手牌非公開
- 観戦ディレイのローカルプレビューを追加
  - 30秒 / 60秒 / 120秒 / 5分
- ディレイ中は公開時刻前のイベントへ進めない
- 牌譜schemaVersionを3へ更新
- viewerSeat / visibilityMode / spectatorDelaySecondsを保存
- 読み込み牌譜のイベント・ブックマークを検証

注意
本番の高レート観戦では、ブラウザ側ではなくサーバー側でディレイと手牌非公開を強制する前提です。
