# Project KSB v0.3.11-1 Fix2

## 修正内容
- 「CPU即リーチ確認」が流局・次局待ちになる不具合を修正。
- CPU手番の設定先を `currentPlayer` ではなく、TurnManagerが実際に参照する `currentSeat` と `turnIndex` に統一。
- デバッグ開始時に `nextRound` と `matchResult` を解除。
- 南家の河・副露・抜き花・リーチ状態を初期化してから即リーチ確認を開始。

## 確認手順
1. `CPU即リーチ確認` を押す。
2. 状態が流局にならず、南家が9pを切る。
3. 南家がリーチし、供託が1増える。
4. 南家の点棒が1000点減る。
5. 宣言牌が横向きになる。
6. 左上のCPU判断欄に南家の打牌・聴牌内容が表示される。
