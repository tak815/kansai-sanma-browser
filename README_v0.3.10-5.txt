Project KSB v0.3.10-5
対局終了・順位判定基盤

実装内容
- 南3局を最終局として扱う暫定対局終了判定
- 南3局で子が和了した場合、対局終了
- 南3局で親が和了した場合、同局連荘
- 南3局で親ノーテン流局の場合、対局終了
- 対局終了時に点棒順で1位〜3位を表示
- 同点時は自家→南家→西家の席順で暫定順位を決定
- デバッグ確認ボタンを追加

確認方法
1. 「南3局 子和了終了」→ 対局結果に順位が表示される
2. 「南3局 親連荘」→ 次局へが有効で南3局のまま本場が増える
3. 「南3局 親ノーテン終了」→ 流局精算後に順位が表示される
4. 対局終了後は「次局へ」が押せない

注意
- 返し点、オカ、ウマ、トビ終了、延長戦条件は未実装です。
- 南3局終了を暫定仕様としており、正式な対局終了条件は今後ルール確定後に拡張します。
