Project KSB v0.3.6B HTML/JS版

Sprint 3-6B: 山・王牌・流動式ドラ表示の土台

変更内容:
- React/Viteなし。index.html一体型のHTML/CSS/JavaScript構成。
- 初期ドラ表示牌を画面表示。
- カン成立時のみ、ドラ表示牌の右側にカンドラ表示牌を追加。
- ドラ表示牌は最大5枚。
- 裏ドラ/カン裏ドラは画面非表示だが内部的に確保。
- カン1回につき通常山から2枚消費（カンドラ表示牌1枚 + カン裏ドラ牌1枚）。
- 嶺上牌は8枚（4トン）で管理。嶺上ツモは表示山残り枚数に含めない。
- カンドラ/カン裏ドラを確保できない、または嶺上牌がない場合はカン不可。

確認ポイント:
1. 画面に初期ドラ表示牌が1枚出ている。
2. デバッグ暗槓 → カンでドラ表示が右に1枚増える。
3. カン後、山残りが2枚減る。
4. カン後、嶺上牌が8枚から7枚になる。
5. 5pではなく5ピン表記・画像を使用。
