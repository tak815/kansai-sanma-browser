## v0.3.8-1 / Sprint 3-8-1: 局進行ステートマシン

- `GamePhase` を追加し、`INIT / DEAL / DRAW / ACTION / DISCARD / NEXT_PLAYER / RYUKYOKU` で局進行を管理。
- `gameState` に局情報、親、現在手番、ターン数、各家の手牌・河・副露・花牌領域を集約。
- `TurnManager` を追加し、東家→南家→西家の順で手番を進行。
- `GameController` を追加し、配牌、ツモ、打牌、次手番、流局の遷移を一元化。
- 既存の花牌ドラ・花牌裏ドラ・カン時のドラ表示牌追加処理は維持。
- CPU手番は暫定的に自動ツモ・ランダム打牌で進行するようにした。

- 初期表示で「あなたの手牌」が空になる不具合を修正。

## v0.3.8-2 / Sprint 3-8-2: 通常ツモ実装
- 自家の通常ツモ→打牌→南家→西家→自家へ戻る基本ループを確認しやすくした。
- デバッグ表示にフェーズ、各家の手牌枚数、各家の河枚数を追加した。
- 南家・西家の河表示を追加し、CPUの自動ツモ打牌を目視確認できるようにした。
- render時に自家手牌参照を再バインドし、状態更新後の空表示を防止した。
