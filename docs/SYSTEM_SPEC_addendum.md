# Project KSB SYSTEM_SPEC 追記 - Sprint 3-7B-1

## 花牌ドラ・花牌裏ドラ判定

ドラ表示牌、カンドラ表示牌、裏ドラ表示牌、カン裏ドラ表示牌が花牌（春・夏・秋・冬）の場合、通常牌のような次牌計算は行わない。

表示牌が花牌だった場合は、春夏秋冬の種類に関係なく、すべての花牌をドラ対象として扱う。

例:

- ドラ表示牌が春 → 春・夏・秋・冬すべてが表ドラ対象
- 裏ドラ表示牌が夏 → 春・夏・秋・冬すべてが裏ドラ対象
- カンドラ表示牌が秋 → 春・夏・秋・冬すべてがカンドラ対象
- カン裏ドラ表示牌が冬 → 春・夏・秋・冬すべてがカン裏ドラ対象

## 重複加算

表ドラ、カンドラ、裏ドラ、カン裏ドラはすべて重複加算する。

例:

- ドラ表示牌が花牌、カンドラ表示牌も花牌の場合、花牌1枚につきドラ2
- さらに裏ドラ表示牌も花牌の場合、花牌1枚につきドラ3
- 複数の花牌表示牌が出た場合も indicator 単位で加算する
