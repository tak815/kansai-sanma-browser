# 卓ルール

- 東南戦
- 配給 35000点
- 原点 40000点
- 返り東あり
- テンパイ連荘
- ノーテン親流れ
- 飛び終了
- 箱下なし
- 沈みウマ -10
- 飛び -10

# 三麻ルール

- 萬子2〜8を使用しない
- チーなし
- 完全先付け
- 面前ダブルバックあり
- ツモ平和なし

## タンヤオ

タンヤオのみでは和了不可。

ただし、

- トイトイ
- 清一色
- 混一色
- 三暗刻
- その他役

との複合は認める。

- 夏
- 秋

- テンパイ牌にもならない
