# 卓ルール

- 東南戦
- 配給 35000点
- 原点 40000点
- 返り東あり（南3局終了時に誰も40000点を超えていない場合）
- 返り東は局終了時に誰かが40000点を超えた時点で終了
- オーラス上がりやめあり
- テンパイ連荘
- ノーテン親流れ
- 飛び終了あり
- 箱下計算あり
- 沈みウマ -10
- 飛び -10

# 三麻ルール

- 萬子2〜8を使用しない
- チーなし
- 完全先付け
- 面前ダブルバックあり
- ツモ平和なし

## タンヤオ

門前タンヤオは単独でも和了可能。

食いタンのみでは和了不可。

ただし、

- トイトイ
- 清一色
- 混一色
- 三暗刻
- その他役

との複合は認める。

# リーチ

- 一発あり
- 見逃し不可
- 1000点でもリーチ可能
- リーチ後のカン可能
- リーチ棒は和了取り
- ダブロン時のリーチ棒は上家取り

# 流局

なし

- 九種九牌
- 四風連打

あり

- 流し役満

---

# ダブロン・飛ばし

- ダブロンあり
- 支払いは放銃者から見て上家 → 下家の順に処理する
- リーチ棒は上家取り
- 箱下まで点数を計算する
- 実際に放銃者を0点以下にした和了者が飛ばし賞を取得する
- 三麻のため三家和・四家立直は存在しない

---

# 四槓流れ

- 4つ目のカン成立時点では流局しない
- 嶺上ツモと打牌まで行う
- その打牌にロンが発生しなかった場合に四槓流れとなる
- 嶺上開花および4槓目後の打牌へのロンを優先する
