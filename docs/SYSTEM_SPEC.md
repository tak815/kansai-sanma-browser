# Project KSB SYSTEM_SPEC

## v0.3.6C 山・王牌仕様

Project KSBは関西三麻仕様として、山・王牌を以下の方針で管理する。

## 1. 嶺上牌

- 嶺上牌は合計8枚、4トンとして扱う。
- 花牌を抜いたときの補充牌は嶺上牌から取得する。
- カン後の補充牌も嶺上牌から取得する。
- 嶺上牌はユーザーに見せる「山残り枚数」には含めない。

## 2. ドラ表示牌・裏ドラ表示牌

- 対局開始時に表示するのは初期ドラ表示牌のみ。
- カンドラ表示牌は最初から固定確保しない。
- カンが入ったときだけ、ドラ表示牌の右側へ1枚ずつ追加する。
- ドラ表示牌は最大5枚。
- 裏ドラ表示牌は画面には表示せず、内部的に確保する。
- カンがない対局では、カンドラ候補牌は通常ツモ可能な山として扱う。

## 3. カン時の枚数処理

カン1回につき内部的には以下を消費する。

- 通常山 2枚
  - カンドラ表示牌 1枚
  - カン裏ドラ牌 1枚
- 嶺上牌 1枚
  - カン後の補充ツモ

ユーザー表示上の山残り枚数は通常山のみを数えるため、カン1回で表示は2枚減る。
嶺上牌の消費は表示上の山残り枚数には影響しない。

## 4. カン不可条件

以下の場合はカン不可。

- ドラ表示牌がすでに5枚ある。
- 新しいカンドラ表示牌とカン裏ドラ牌を確保できない。
- 嶺上牌が残っていない。

## 5. 表示方針

デバッグ中は嶺上牌8枚を横一列で表示する。
消費済みの嶺上牌は白い牌ではなく、空き枠として表示する。
正式版では4トン表示へ切り替える。
