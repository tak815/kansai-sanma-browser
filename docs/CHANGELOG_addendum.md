# CHANGELOG 追記 - Sprint 3-7B-1

## Sprint 3-7B-1 修正版

### Fixed

- ドラ表示牌・裏ドラ表示牌が花牌だった場合に、通常牌と同じ次牌計算へ流れてしまう可能性を修正。
- 花牌表示牌の場合は春夏秋冬すべてをドラ対象にする専用分岐を追加。
- 表ドラ、カンドラ、裏ドラ、カン裏ドラで共通利用できるドラ判定関数へ整理。
- 花牌ドラ・花牌裏ドラが重複加算される仕様を明文化。

### Test

- 表示牌「春」で春夏秋冬すべてがドラ対象になることを確認。
- 表示牌「夏」など別の花牌でも春夏秋冬すべてが対象になることを確認。
- 花牌表示牌が複数ある場合、花牌1枚に対して複数ドラとして加算されることを確認。
- 通常牌の次牌ドラ判定は維持されることを確認。
