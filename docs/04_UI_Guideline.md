# 目的

# デザインコンセプト

目標

「雀魂の見やすさ」

「天鳳の軽さ」

「関西三麻専用UI」

・情報は少なく

・牌は大きく

・盤面を見やすく

・クリックしやすく

・長時間遊んでも疲れない
