# 関西三麻ブラウザゲーム 開発計画書

Version 1.0

---

# プロジェクト目標

ブラウザで遊べる本格的な関西三麻ゲームを制作する。

目標は試作品ではなく、
長期間運営できるオンライン麻雀ゲームとする。

---

# 開発方針

- HTML
- CSS
- JavaScript

のみ使用する。

ライブラリには依存しない。

GitHubでバージョン管理を行う。

RuleBook.md をルールの正とする。

完成したファイル単位で管理する。

---

# UI方針

雀魂を参考にした見やすいUI。

ただし重くなりすぎず、
ブラウザで軽快に動作するデザインを採用する。

関西三麻専用レイアウトを採用。

・河は7×3表示

・下家・上家は横向き表示

・副露は左方向へ追加

・花牌は副露の上へ表示

・牌山は雀魂風デザイン

# 品質方針

コードは保守性を重視する。

UIは先に完成させる。

機能追加でUIを大きく変更しない。

完成後もアップデートしやすい設計を維持する。

# 最終目標

「関西三麻ならこのゲーム」

と言われるブラウザゲームを作る。
