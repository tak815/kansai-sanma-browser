Project KSB v0.3.9-2 / Sprint 3-9-2
リーチ判定・リーチ宣言基盤

変更内容:
- テンパイ時のみリーチボタンを有効化
- 花牌保有中はリーチ不可
- 副露・カン表示がある場合はリーチ不可
- リーチ宣言時に供託を+1
- リーチ宣言牌を河で横向き表示
- リーチ後はツモ切りのみ許可
- リーチ済み時のみ裏ドラを有効化
- デバッグ「リーチ確認」を追加

確認手順:
1. リーチ確認を押す
2. リーチボタンが有効になることを確認
3. リーチを押す
4. 供託が1になることを確認
5. 好きな牌を打牌して宣言牌が横向きになることを確認
6. 次巡以降はツモ切り以外を捨てられないことを確認
