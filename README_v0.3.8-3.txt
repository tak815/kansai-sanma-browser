Project KSB v0.3.8-3 / Sprint 3-8-3
抜き花補充（任意タイミング版）

変更内容:
- 自家の「抜き花」ボタンを追加。
- 花牌は空気牌として扱い、捨てられないように制御。
- 自家はツモ後・打牌前に任意で抜き花可能。
- AI（南家・西家）は花牌を持った場合に即抜き。
- 抜き花後は嶺上牌から補充。
- 抜き花補充では通常山残り表示は減らず、嶺上牌残りだけ減る。
- 自家の抜き花表示エリアと、デバッグ用の抜き花枚数表示を追加。
- 「抜き花確認」デバッグボタンを追加。

確認項目:
1. 「抜き花確認」を押す。
2. 手牌に春がある状態で「抜き花」ボタンを押す。
3. 春が抜き花エリアへ移動する。
4. 嶺上牌が1枚減る。
5. 山残り表示は減らない。
6. 花牌をクリックして捨てようとすると捨てられない。
7. 通常牌はこれまで通り打牌できる。
8. 南家・西家が花牌を持つと自動で抜き花し、自家へ戻る。
